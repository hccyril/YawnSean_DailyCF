**提示 1：** 先凑出来比较接近的数。

**提示 2：** 剩余的进行微调。

我们一开始先塞入一堆 $0$ ，这样用 $x$ 个数就能产生 $C_x^3$ 个有效的三元组。

剩下了多少呢？一定剩下的不超过 $C_{x+1}^3-C_x^3$ 个，不然我们可以把 $x$ 增大 $1$ 。

接下来剩下的就只有 $x^2$ 量级的了，也就是 $\mathcal{O}(k^{2/3})$ 左右的大小了，但我们也不能再往里面塞 $0$ 了。

接下来一个比较简单的策略是塞 $a,b,c$ 满足 $a+b+c=0$ ，这样方案数就是它们相乘。只要 $a,b,c$ 不可能跟别的元素配对就行。这件事很容易办到，你直接用 $(1,3,-4)$ 和它的倍数（如 $10,100,\dots$ 倍）即可。不断凑就行，这里剩下的并不多，很快就能凑出来。

时间复杂度为 $\mathcal{O}(k^{1/3})$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    k = II()
    
    for i in range(10000):
        if i * (i - 1) * (i - 2) // 6 > k:
            i -= 1
            k -= i * (i - 1) * (i - 2) // 6
            ans = [0] * i
            break
    
    cur = 1
    while k:
        v = math.isqrt(k)
    
        ans.append(4 * cur)
        ans.extend([-3 * cur] * v)
        ans.extend([-cur] * v)
    
        k -= v * v
        cur *= 10
    
    print(len(ans))
    if ans: print(*ans)
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int k;
	cin >> k;

	vector<int> ans;

	for (int i = 0; i < 10000; i ++) {
		if (1ll * i * (i - 1) * (i - 2) / 6 > k) {
			i --;
			k -= 1ll * i * (i - 1) * (i - 2) / 6;
			while (i --) ans.emplace_back(0);
			break;
		}
	}

	int cur = 1;
	while (k) {
		int v = sqrt(k);
		if (v * v > k) v --;

		k -= v * v;
		ans.emplace_back(4 * cur);
		while (v --) {
			ans.emplace_back(-3 * cur);
			ans.emplace_back(-cur);
		}

		cur *= 10;
	}

	cout << ans.size() << '\n';
	for (auto &x: ans) cout << x << ' ';

	return 0;
}
```
