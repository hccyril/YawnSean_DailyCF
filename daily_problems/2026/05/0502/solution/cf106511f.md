**提示 1：** 先确定角，再一步步推。

首先，如果行或列是 $1$ 是很容易讨论的情况，因为相当于是一个序列，只可能选择反转或者不反转。

否则，考虑角的一个数，并从之延申。

因为角上的点只有两个其他的点跟它距离为 $1$ ，而这样的点总共只有 $4$ 个，因此只有四种选择。

而接下来考虑和它相邻的点是按照什么方式来排的，而一旦这件事确定，剩余元素的位置也都确定了，因为 $3$ 个点足以对每个元素定位。

发现 $n=m$ 时可以按照任意顺序，而 $n\neq m$ 时只有一种可能，不然的话就没办法在原来的矩阵填下了。（考虑对角的那个位置就可以推出矛盾）

所以两种情况分别答案是 $8,4$ 。

时间复杂度为 $\mathcal{O}(1)$ 。

### 具体代码如下——

Python 做法如下——

```Python []
def main():
    t = II()
    outs = []
    
    for _ in range(t):
        n, m = MII()
        
        if n == 1 and m == 1: outs.append('1')
        elif n == 1 or m == 1: outs.append('2')
        elif n != m: outs.append('4')
        else: outs.append('8')
    
    print('\n'.join(outs))
```

C++ 做法如下——

```cpp []
int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);

	int t;
	cin >> t;

	while (t --) {
		int x, y;
		cin >> x >> y;

		if (x == 1 && y == 1) cout << 1;
		else if (x == 1 || y == 1) cout << 2;
		else if (x != y) cout << 4;
		else cout << 8;

		cout << '\n';
	}

	return 0;
}
```
